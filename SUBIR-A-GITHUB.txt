BRAWL META - ZIP DEFINITIU

COM PUJAR-HO A GITHUB + VERCEL

1) Descomprimeix aquest zip.
2) A GitHub, entra al repo brawlmeta.
3) Esborra o substitueix els fitxers antics.
4) Puja ELS FITXERS I CARPETES DE DINS del zip, no el zip sencer.
5) Fes Commit changes.
6) Vercel farà deploy automàtic.
